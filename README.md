# awsome-mirai
本仓库用于收集mirai相关社区应用，例如基于mirai的bot，mirai—console的插件以及使用了其他语言sdk的项目
如果您有这类项目，欢迎提交pull request将您的项目添加到这里(注意，本仓库仅接受开源项目的仓库地址，不接受二进制或未托管到源代码管理的项目)
